# Headers

Header files have the extension `.h` and are technically fully valid C files.

Any C code could be written in them, but they serve a particular purpose: **They declare functions you can use in your own .c file(s)**.

For example, let's say we have the following function in a file named `sum.c`:
```c
int sum (int a, int b) {
    return a + b;
}
```
And let's say we now want to use this function in our `main.c` file.

Instead of having to copy the function over, we can make a `sum.h` file that declares the function `sum`:
```c
int sum(int a, int b);
```
Now we include `sum.h` in our `sum.c`:
```c
#include "sum.h"
int sum (int a, int b) {
    return a + b;
}
```
And most importantly, in our `main.c`:
```c
#include "sum.h"
```
And there we go! We can now use the `sum` function in `main.c`!

## Standard Library Headers
A *library* contains many different functions. One that is almost always pre-installed is the so-called *standard library*.

It contains many different important functions you can use, like `printf` for printing out messages to the console and `scanf` for reading messages from the console. (More on them [here](1.6%20printf%20and%20scanf.md).)

These functions are declared in header files you can include very similarly to your own. The key difference is that you usually type the header file into `<>` brackets:
```c
#include <stdio.h>  // Standard library header
#include "custom.h" // Custom header
```
*Note: You can technically use both `<>` and `""` for standard library headers, but the convention is to distinguish them.*

### Some standard library headers
You can find a full list of C standard library headers [here](https://cppreference.com/c/header).

I want to go over a few of them here in detail, because these will be important in the future:
#### `<stdio.h>`: Library for input/output functions
- `printf` and its variants
- `scanf` and its variants
- File Management
- ...
#### `<stdlib.h>`: Library for general utilities
- `malloc` and `free` (and others) for memory management
- `strtol` and similar functions for type conversion
- `system` for running a command in the terminal
- `abs` and similar functions for getting the absolute value of an int
- ...
#### `<string.h>`: Library for working with strings
- `strcpy` and similar functions for copying/manipulating strings
- `strlen` and others for getting a string's length
- `strcmp` for comparing two strings
- `memcpy` and others for memory management
- ...
#### `<math.h>`: Library for mathematics and arithmetics
- `abs` and `fabs` for computing the absolute value of an int/float
- `log` and similar functions for logarithmic calculations
- `pow` and `sqrt` and similar functions for calculating powers and square roots
- `sin`, `cos`, `tan`, ...
- `round`, `floor` and `ceil` for rounding floats
- ...
