# Operators, if and Ternary

## Arithmetic
These are your typical mathematical operations:
```c
int a = 5;
int b = 2;
a + b; // 7
a - b; // 3
a * b; // 10
a / b; // 2
```
There is also the modulo operator:
```c
a % b; // 1
```
This divides a by b and returns the remainder of that division operation.

If you're a math nerd, it returns `r` for which:
```
a = n*b + r (with n ≠ 0 and |r| < |n|)
```
### Special form: Increment (`++`) and decrement (`--`)
The increment and decrement operators are a short form of adding/subtracting 1 from a variable:
```c
int test = 3;
test++; // equivalent to test = test + 1
test--; // equivalent to test = test - 1
```

## Assignment
You have already seen the basic assignment operator, the single `=`.
This assigns a value to a variable.
You can, however, also combine it with an arithmetic or a bitwise operation to apply it to the variable's value, for example:
```c
int a = 10;
a += 5; // equivalent to a = a + 5
```

## Comparison
Comparison operators compare two different values and evaluate to either `1` if the comparison is `true` or `0` if the comparison is `false`.
#### `==` equals
```c
int a = 10;
a == 10; // true
```
#### `!=` not equal
```c
int a = 10;
a != 10; // false
```
#### `>` greater than
```c
int a = 10;
a > 5; // true
```
#### `>=` greater than or equals
```c
int a = 10;
a >= 10; // true
```
#### `<` less than
```c
int a = 10;
a < 5; // false
```
#### `<=` less than or equals
```c
int a = 10;
a <= 5; // false
```
## Logical
These operators combine multiple expressions together.
#### `&&` AND
```c
int a = 10;
a == 9 && a > 5; // false
```
#### `||` OR
```c
int a = 10;
a == 9 || a > 5; // true
```
#### `!` NOT
```c
int a = 10;
!(a == 10); // false
```
*Note: `&&` and `||` only evaluate the right expression if the left expression doesn't already determine the result.*
```c
false && /* will not be evaluated, result is false */
true  || /* will not be evaluated, result is true */
```


## If Statements and Ternary
These comparisons can either be used as a `bool` like this:
```c
#include <stdbool.h>
bool isGreater = 10 > 5; // true
```
To actually execute different code depending on whether an expression is `true` or false, we can use `if` statements:
```c
if (10 > 5) {
    // Code to execute if true
} else {
    // Code to execute if false
}
```
You can put anything in an if statement that evaluates to a number, so that includes all logical expressions and even actual numbers:
```c
int test = 3;
if (test) {
    // Code to execute if test != 0
} else {
    // Code to execute if test == 0
}
```
As you can see, when a variable is passed in, it evaluates to `true` if its value is not 0. You can also use the not operator to reverse the behavior:
```c
int test = 3;
if (!test) {
    // Code to execute if test == 0
} else {
    // Code to execute if test != 0
}
```
### Ternary operators
Ternary operators can be used as a shorthand to pick what value to return on what conditions.
```c
int val = (10 > 5 ? 3 : 4); // if 10 > 5, then val is 3, otherwise val is 4
```

## *Extra: Bitwise*
These operators work on a binary level:
#### `&` AND
The bitwise and operator compares every bit of two numbers against each other. If both bits at a given position are 1, the resulting bit is 1. In all other cases it is 0.
```c
0b01001110 & 0b00100101;// == 0b00000100
// is equivalent to
78 & 37;                // == 4
```
#### `|` OR
Similar to the `&` operator, but now the resulting bit is only 0 when both bits are 0. If either or both bits are 1, the resulting bit is 1.
```c
0b01001110 | 0b00100101;// == 0b01101111
// is equivalent to
78 | 37;                // == 111
```
#### `^` XOR (exclusive OR)
Similar to the `|` operator, except now the resulting bit is only 1 if the two bits are different. If they are both 0 or 1, the resulting bit is 0.
```c
0b01001110 ^ 0b00100101;// == 0b01101011
// is equivalent to
78 ^ 37;                // == 107
```
#### `~` NOT
Inverts (or "flips") every bit:
```c
~0b01001110;    // == 0b10110001
// is equivalent to
~78;            // == -79 (signed) == 4294967217 (unsigned)
```
#### `<<` Left Shift
Shifts a binary number left by a specified amount of digits. This is equivalent to multiplying that number by 2^n.
```c
0b101 << 2;     // == 0b10100
// is equivalent to
5 << 2;         // == 20
// is equivalent to
5 * 2 * 2;      // == 20
```
#### `>>` Right Shift
Shifts a binary number right by a specified amount of digits. This is equivalent to dividing that number by 2^n.
```c
0b101 >> 1;     // == 0b10
// is equivalent to
5 >> 1;         // == 2
// is equivalent to
5 / 2;          // == 2
```
*Excursion: A right shift will "sign-extend" your number. New bits inserted at the start are either 0 if the type is `unsigned`, or it will copy the left-most bit if the type is `signed`. This has to do with how negative numbers are represented.*
