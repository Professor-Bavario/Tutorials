# Our first program

If you compiled and ran [0.3.1.c](../0%20-%20Setup%20and%20Introduction/exercises/0.3.1.c), you will have seen this as our code:

```c
// EXERCISE 0.3.1

#include <stdio.h>

int main () {
    printf("Hello, welcome to this tutorial by Bavario!\n");
    return 0;
}
```

We will now go through this program 1 by 1 and I will explain exactly what each statement means.

These will each be explained in more depth in the other `.md` files in this folder, so make sure you check these out as well.

---
```c
// EXERCISE 0.3.1
```
This is a **comment**. These can be left in by the programmer and will never be read by the compiler.

You can add a comment by writing it after `//` somewhere. Everything in the line after the two slashes is a comment.

```c
printf("Bringle"); // This prints a name
```

You can also add comments that span multiple lines using `/*` at the beginning and `*/` at the end:

```c
/* This is a
multiline
comment!
*/
```
---
```c
#include <stdio.h>
```
This includes a **header**, in this case a **standard header**.

A header defines functions that you can use in your code. In our case, we include the **standard input/output** header, so that we can use the function `printf` later on in the code.

---
```c
int main () {
```
This is defining a function with a return type, a name, its arguments, and a body.

`int` is the integer type, which means an integer will be returned after the function finishes.

`main` is the name of the function.

Inside the brackets `()`, arguments can be specified that need to be passed to the function. In this case we don't provide any, which means **anything** can be passed to it - we do not care.

The opening curly bracket `{` now specifies the start of the **function body**. The body contains the code that should be executed inside of a function. It is closed again later with `}`.

**`main` in this case is a special function: It is called by the Operating System when our program starts. So this is the first function that will be called.**

---
```c
printf("Hello, welcome to this tutorial by Bavario!\n");
```
This is a classic function call.

We first name the function we want to call - in this case `printf`. (`printf` is defined in `<stdio.h>` and prints something to the console.)

Then, inside of the of the brackets `()`, we specify the arguments we want to pass to this function. In our case, we pass a string that we denote with `""`. `printf` actually supports an endless amount of arguments. For more, read [1.4 printf]().

**After every statement, a `;` has to be written.**

---
```c
    return 0;
}
```
This tells our `main` function to return the integer value `0` and then we close the function body.

For `main`, the value we return is the **program status**. If we return `0`, it means everything worked. If we don't, that means something went wrong and you can use the status returned to figure out what the issue is.
