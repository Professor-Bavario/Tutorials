# Why are we learning C?

C is a **general-purpose** programming language that was created in 1972. Back in the day, its syntax was so revolutionary that it inspired almost all of the later major programming languages like C++, Java, C#, JavaScript, Python, ...

Nowadays it is still widely used because of how close it allows you to program to the hardware level, without having to learn Assembly. Because of this, it has very good performance, which also makes it a good choice for making games (*for example: The Paper Mario series*) or other software like OBS.

More important than that: C is also to a big part the programming language used by operating systems like Linux and Windows, simply because how much control it gives you and how performant it is.

**The main reason why *we* are learning it is because it's still a rather simple language, and once you learn its syntax, I believe you will be set for essentially any other programming language out there.**

## Later on: C++

C++ was released in 1985 as an extension to C. Almost every program you write in C will work and be identical in C++. (*I say almost because there are some differences, more on that later.*)

The main benefit of C++ is that it adds **Object-oriented programming** (OOP). We will get into what exactly it is later, but this is a very important principle in Programming.

If you want to make a video game, C++ is traditionally the go-to choice. Nintendo uses it as its standard language for making games.

**The idea of this tutorial is that C will introduce you to basic programming and its concepts, and then C++ will expand on it with OOP. And then hopefully, by the end of this, you will be able to code!**
